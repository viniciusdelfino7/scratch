Guitar Hero (Projeto Makey Makey)

Jogo criado no Scratch baseado no clássico Guitar Hero. Desenvolvido com dedicação pelos alunos Caio Henrique e Livia Cardia.

O jogo foi projetado para ser jogado de duas formas distintas:

Opção 1 - Placa Makey Makey

Interação completa utilizando hardware externo:

  • Placa Makey Makey
  • Cabo USB
  • Cabos jacaré
  • Massinha de modelar

Opção 2 - Utilizar Setas

Controles padrão mapeados no teclado:

  • ↑ Pra cima — Vermelho
  • ↓ Pra baixo — Amarelo
  • → Direita — Azul
  • ← Esquerda — Verde
